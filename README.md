# Projeto-EDA_mais_serie_temporal_com_perceptron_multicamadas_MLS
Neste projeto iremos fazer uma análise exploratória do consumo de gasolina no estados do Brasil entre os anos de 1990 à 2020 em seguida fazer uma projeção(Forcast) para os próximos anos.  Multilayer Perceptron (MLP) A perceptron multicamadas (MLP) é uma rede neural semelhante à perceptron, mas com mais de uma camada de neurônios em alimentação direta. Tal tipo de rede é composta por camadas de neurônios ligadas entre si por sinapses com pesos. O aprendizado nesse tipo de rede é geralmente feito através do algoritmo de retro-propagação do erro, mas existem outros algoritmos para este fim, como a Rprop.
